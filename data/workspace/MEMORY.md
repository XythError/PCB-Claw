# Long-term Memory

This file stores persistent facts and context remembered by PCB-Claw.
It is updated automatically when the user issues /remember <text>
or when the LLM calls the memory tool with op=write.

## How to use
- User command: /remember <fact>   → appends a fact here
- User command: /note <text>       → appends to today's daily note
- LLM tool call: memory write      → agent writes a fact autonomously
- Read memory: /memory or memory read tool
